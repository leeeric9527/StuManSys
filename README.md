# StuManSys
数据库课程设计：学生-课程-老师管理系统
## 环境
语言：python3.7.2

第三方库：web.py 0.40-dev1、psycopg2 2.7.6.1
## 文件结构
init.py:初始化程序，用于初始化数据库。

config.py:配置文件，记录配置的设定。

StuManSys.py:主程序入口

file.py:将html文件、css文件等读入到内存的程序
